## 4. 深度分析

### 4.1 L4 的"闪电战"风格

L4 的 AvgLen (75.1) 远低于其他模型 (81.6 ~ 87.3)，这不是偶然：

```
各模型平均 AvgLen:
  L0: 81.6  (弱，被终结快)
  L3: 84.0  (中)
  L1: 86.6  (长拉锯型)
  L2: 87.3  (最长，策略犹豫)
  L4: 75.1  ← 最短，且是最强的
```

传统理解：弱方 AvgLen 短（因为被终结快）。但 **L4 既是最强又是最短**。
这说明 L4 学会了一种**高效进攻策略**——不需要漫长的拉锯就能赢。

对比 L3 vs L4 的两个方向：
- L4 → L3: WR=87%, AvgLen=74.3 (L4 快速击败 L3)
- L3 → L4: WR=27%, AvgLen=68.3 (L3 输得更快)

L3 面对 L4 时不仅输了，而且输得最快 (68.3 步)。
L4 赢 L3 需要 74.3 步，比 L3 输给 L4 的 68.3 步更长——
说明 L4 赢的时候是"主动围杀"(稍需布局)，L3 输的时候是"直接崩盘"(快速犯错)。

### 4.2 实力层级分化

根据 L4 的加入，模型实力形成了清晰的三个层级：

| 层级 | 模型 | 平均胜率 | 特征 |
|------|------|----------|------|
| **S 级** | L4 | 80.3% | 闪电战风格，全面碾压 |
| **A 级** | L3, L1 | 61~62% | 稳健型，互有胜负 |
| **B 级** | L2 | 49.8% | 专精化失败，低于平均 |
| **C 级** | L0 | 25.5% | 基础模型，全面落后 |

L4 和 A 级之间存在 **~18% 的胜率断层**，这是目前最大的代际跃升。

### 4.3 L1 vs L3 的"平级悖论"

一个有趣的发现：L1 (61.0%) 和 L3 (62.0%) 平均胜率几乎相同，但它们的对战结果不对称：
- L3 → L1: **71%** (L3 大幅领先)
- L1 → L3: **55%** (L1 仅略微领先)

L3 单挑 L1 有明显优势，但 L3 被 L4 碾压得更惨 (27% vs L1 的 42%)。
这说明 **L3 的策略体系对 L4 特别脆弱**，可能是因为 L4 直接从对 L3 的训练中学到了针对性打法。

- Loss 收敛更深 (0.238 vs 0.503)
- 对所有对手的平均胜率高出 18%

## 5. L4 综合评价

### 优势
1. **绝对实力最强** — 对所有对手胜率 ≥ 62%，平均 80.3%
2. **高效进攻风格** — AvgLen 75.1，最短，赢得快
3. **训练效率高** — 30K 局就达到了远超 L3 的水平
4. **Loss 收敛优秀** — 0.238，说明策略学习充分

### 潜在风险
1. **对 L1 的胜率偏低 (62%)** — L1 的稳健拉锯风格可能部分克制 L4 的速攻
2. **策略单一性** — AvgLen 极低可能意味着策略不够多样，遇到防守型对手可能受阻
3. **架构天花板** — 仍然是 10×64，容量限制可能在更高水平显现

## 6. L5 训练设想

### 6.1 核心问题

L4 的短板在于 vs L1 只有 62%。L1 是一个"稳健拉锯型"选手，说明 L4 的速攻策略在面对不急于进攻的防守型对手时效果打折。L5 的目标是在保持进攻效率的同时，**提升对防守型对手的适应能力**。

### 6.2 推荐训练参数

| 参数 | 值 | 理由 |
|------|-----|------|
| 架构 | 10×64 | 保持一致，控制变量 |
| 学习率 | **3e-4** | 进一步降低，追求更精细的策略调整 |
| 训练对手 | **L4** | 阶梯式继续 |
| 目标胜率 | **80%** | 与前代一致 |
| Parallel Games | 80 | 与 L4 一致 |
| Seed | 42 | 可复现 |

### 6.3 备选方案：opponent-mix 训练

如果纯 vs L4 训练出现 L2 式的"专精化"问题，考虑混合训练：

```bash

uv run python src/train.py --num-blocks 10 --num-filters 64 \
  --learning-rate 3e-4 --target-win-rate 0.80 \
  --eval-opponent L4 --parallel-games 80 --seed 42
```

### 6.4 L5 成功标准

1. **矩阵平均胜率 ≥ 83%** (超过 L4 的 80.3%)
2. **vs L1 ≥ 70%** (修补 L4 的短板)
3. **AvgLen 不高于 80** (保持进攻效率)
4. **训练局数 ≤ 30K** (保持训练效率)
